| Command                                         | Explanation                                                    |
| ----------------------------------------------- | -------------------------------------------------------------- |
| `source.venv/bin/activate`                     | Activate the virtual environment (Linux/macOS)                 |
| `.venv\Scripts\activate`                        | Activate the virtual environment (Windows)                     |
| `pip install package_name`                      | Install a specific package into the virtual environment        |
| `pip install --upgrade package_name`            | Update a specific package to the latest version                |
| `pip freeze > requirements.txt`                 | Save all installed packages and versions into requirements.txt |
| `pip install -r requirements.txt`               | Install all packages listed in requirements.txt                |
| `pip install -r requirements-dev.txt`           | Install all dev tools listed in requirements-dev.txt           |
| `pip install --upgrade -r requirements-dev.txt` | Update all dev packages to the latest versions                 |
